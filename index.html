this is the 3rd site i made to show that i love 00111000 01100010 01101001 01110100 too.

the first two sites were a bit too busy, so i spent some time stripping away. less is more in design.

i've made use of many standard libraries (HTML5 boiler plate, Bootstrap 3, Font-Awesome, Skrollr, NiceScroll.)

so please take a look at the code, you'll notice the 8bit header font is not an actual font. It was inspired by your logo, and is generated on page load using data attributes, a custom 2D alphabet array, for loops, and a HSL colour generator which breaks the colours down into various ranges, and generates random colours therein.

i've also used the Skrollr library for the parallax effect, (it was a lot busier, but less is more,) and the on-scroll colour changes to the 3 in the bottom right corner.

i have also written large part of it one the road, and with dodgy internet thanks to load shedding. so i hope it will suffice.
